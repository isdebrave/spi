## 📌 실행 방법

1. 깃헙 클론
   ```
   git clone https://github.com/isdebrave/spi.git
   ```
2. 프론트엔드 실행
   ```
   cd frontend
   npm i
   npm run dev
   ```
3. 백엔드 실행
   ```
   cd backend
   npm i
   npm run start
   ```
